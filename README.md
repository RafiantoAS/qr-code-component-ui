# QR Code Component UI

A modern QR Code Component built using pure HTML and CSS.

This project focuses on creating a clean, responsive, and visually appealing card interface that displays a QR code along with supporting content. The design emphasizes layout structure, spacing, typography, and component styling while maintaining a pixel-perfect appearance.

Developed as part of a frontend practice challenge to strengthen core web development skills and improve understanding of modern UI implementation.
